# 🤖 AI Insurance Claim Automation System

A full-stack AI/ML web application that predicts insurance claim amounts and detects fraudulent claims using Machine Learning models integrated with a Flask web interface.

---

## 🚀 Features

✅ Claim Amount Prediction (Regression)  
✅ Fraud Detection (Classification)  
✅ Combined UI for both functionalities  
✅ Interactive and modern frontend (HTML, CSS, JS)  
✅ Flask-based backend API  

---

## 🧠 Machine Learning Models

### 💰 Claim Prediction
- Type: Regression  
- Algorithms: Linear Regression / Random Forest (or your model)  
- Target: `charges`

### 🚨 Fraud Detection
- Type: Classification  
- Algorithm: Random Forest Classifier  
- Target: Fraud / Not Fraud  

---

## 📊 Datasets Used

### 1️⃣ Insurance Dataset (`insurance.csv`)
- age  
- sex  
- bmi  
- children  
- smoker  
- region  
- charges (Target)

### 2️⃣ Claim Fraud Dataset (`claim_data.csv`)
- Billed Amount  
- Paid Amount  
- Insurance Type  
- Claim Status  
- Outcome (Target)
